Le Ciel de Neptune

Nous allions nous baigner dans la lumière de Lune
Se sentant caresser de nos brasses d’argent
S’amusent à briller planctons luminescents
Imitant leurs aînées du ciel de Neptune

Nager dans l’océan
Froid mais doux
Puis se tenir debout
Devant le feu brûlant
